Chapter 7: Control Flow

if Expressions

In Midori, if is an expression, not a statement. It always returns a value, and both branches must produce the same type. The optional elif keyword chains additional conditions without nesting.

let status = if temperature > 30 {
    "hot"
} elif temperature > 20 {
    "warm"
} else {
    "cool"
}

match Expressions

The match expression is Midori's powerful pattern matching construct. It compares a value against a series of patterns and executes the arm that matches. Match arms can destructure data, bind variables, and include guards.

match value {
    0                  => "zero",
    1 | 2              => "small",
    n if n < 10       => "less than ten",
    _                   => "other",
}

For Loops

The for loop iterates over any iterable value. Midori's standard library provides ranges (0..n for exclusive, 0..=n for inclusive) and collection iterators.

// Range-based for loop
for i in 0..5 {
    println("count {i}")
}

// Iterate over a list
for item in items {
    println(item)
}

// With index and value destructuring
for (i, val) in enumerate(items) {
    println("[{i}] = {val}")
}

While Loops

The while loop repeats a block as long as a condition is true. It is the simplest looping construct and is ideal when the number of iterations is not known in advance.

let mut n = 100
while n > 1 {
    n = if n % 2 == 0 { n / 2 }
         else          { n * 3 + 1 }
    println(n)
}

Loop and break/continue

The bare loop creates an infinite loop. Use break to exit and continue to skip to the next iteration. Both break and continue work in for and while loops as well. break can return a value from a loop.

let mut guess = 0
let answer = loop {
    guess = guess + 1
    if check(guess) {
        break guess  // loop returns the value
    }
    if guess > 100 {
        break 0     // default value
    }
}

// Loop labels for nested control flow
'outer: for x in 0..10 {
    for y in 0..10 {
        if x * y == 42 {
            break 'outer
        }
    }
}
