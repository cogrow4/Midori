Chapter 8: Structs and Enums

Structs

Structs let you group related data together into a named type. Each field has a name and a type. Structs are defined with the struct keyword and created with named field syntax. Fields are accessed with dot notation.

struct User {
    name: Str,
    email: Str,
    age: Int,
}

let user = User {
    name: "Alice",
    email: "alice@example.com",
    age: 30,
}

println(user.name)  // "Alice"

Tuple Structs

For simple wrappers, you can use tuple structs where fields are positional rather than named. These are useful for newtype patterns or when field names would be redundant.

struct Point(Int, Int)
struct Color(Int, Int, Int)

let p = Point(3, 7)
let c = Color(255, 128, 0)
println(p.0)  // 3, accessed by position

Enums

Enums represent a type that can be one of several variants. Each variant can optionally carry associated data. Enums are the backbone of Midori's error handling and pattern matching.

enum Shape {
    Circle(Float),        // radius
    Rectangle(Float, Float),  // width, height
    Triangle(Float, Float),  // base, height
}

fn area(shape: Shape) -> Float {
    match shape {
        Circle(r)       => 3.14159 * r * r,
        Rectangle(w, h) => w * h,
        Triangle(b, h)  => 0.5 * b * h,
    }
}

Methods with impl

Methods are functions associated with a type, defined in an impl block. The first parameter is always self, which can be taken by value, by reference (&self), or by mutable reference (&mut self).

struct Counter {
    value: Int,
}

impl Counter {
    fn new() -> Self {
        Self { value: 0 }
    }

    fn increment(mut self) {
        self.value = self.value + 1
    }

    fn current(self) -> Int {
        self.value
    }
}

let mut c = Counter::new()
c.increment()
println(c.current())  // 1

Enum with Methods

Enums can also have methods via impl. This is a powerful way to attach behavior to sum types.

impl Shape {
    fn scale(self, factor: Float) -> Shape {
        match self {
            Circle(r)       => Circle(r * factor),
            Rectangle(w, h) => Rectangle(w * factor, h * factor),
            Triangle(b, h)  => Triangle(b * factor, h * factor),
        }
    }
}
