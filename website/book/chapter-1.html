Chapter 1: Introduction

What is Midori?

Midori is a compiled programming language built from scratch in Rust. It features Python-inspired syntax with C-style braces, type inference, string interpolation, pattern matching, and method dispatch. Midori compiles through C to native code via Clang or GCC.

The name Midori — Japanese for "green" — reflects the project's green field design: a fresh take on a compiled language without legacy constraints.

Design Philosophy

Midori is guided by a few core principles:

  • Simplicity. No semicolons, no ceremony. Top-level code just runs.
  • Approachable syntax. GDScript-inspired with braces. Functions use func, blocks use { }.
  • Strong types, inferred. Static typing with full type inference — write less, catch more.
  • Built from scratch. Lexer, parser, type checker, codegen — all hand-written in Rust.

Hello, Midori

A complete Midori program in one line:

// hello.mi
print("Hello, World!")

Functions use the func keyword:

func greet(name) {
    print("Hello, {name}!")
}

greet("World")
