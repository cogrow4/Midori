Chapter 14: The Standard Library

Midori ships with a comprehensive standard library that covers the essentials. The library is organized into modules following the same conventions as user code. Everything in the standard library is available by default — no explicit imports needed for core types.

Core Types

The core module provides the fundamental types we have used throughout this book: Int, Float, Bool, Str, Char, Option, and Result. It also includes commonly used functions for type conversion and basic operations.

// Type conversion utilities
let n: Int   = parse_int("42")
let s: Str   = to_string(42)
let f: Float = to_float(42)

// String utilities
let upper = uppercase("hello")
let lower = lowercase("WORLD")
let parts = split("a,b,c", ",")
let len   = len("hello")

Collections

The collections module offers dynamically-sized data structures. The most commonly used are List<T> (a growable array), Map<K, V> (a hash map), and Set<T> (a hash set). All collections implement standard iteration interfaces.

let mut list = [1, 2, 3]
list.push(4)
println(list.pop())  // 4

let mut map = Map::new()
map.insert("key", 42)
println(map.get("key"))  // Some(42)

let set: Set<Int> = [1, 2, 3].into()

Iteration

The iter module provides combinators for functional data processing. These integrate with the pipe operator |> for clean transformation pipelines.

let result = [1, 2, 3, 4, 5]
    |> filter(fn(x) { x % 2 == 0 })
    |> map(fn(x) { x * x })
    |> collect<List>()
// [4, 16]

Input/Output

The io module provides file and console I/O. It includes File for file operations, stdin/stdout/stderr for standard streams, and convenience functions like read_to_string.

// Read an entire file
let contents = read_to_string("data.txt")?

// Write to a file
let mut file = File::create("output.txt")?
file.write("Hello, file!")?

// Read user input
let input = read_line()?
println("You said: {input}")

Common Patterns

The standard library also includes a time module for working with dates and durations, a math module for mathematical functions, and a json module for JSON parsing and serialization.

use time::{Duration, Timer}
use math::{sqrt, sin, cos, pi}

let timer = Timer::start()
// ... expensive operation ...
println("Elapsed: {timer.elapsed()}")

// JSON parsing
let data = parse_json("{\"name\": \"Midori\"}")?
println(data["name"])

This concludes our tour of the Midori standard library, and this book. You now have the knowledge to build real applications in Midori. The standard library documentation provides full details on every module — refer to the docs for the complete reference.
