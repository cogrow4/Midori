The Midori Programming Language

A comprehensive guide to writing beautiful, fast, and safe code in Midori.

Table of Contents

  1. Introduction — What is Midori? Design philosophy. Who this book is for.
  2. Getting Started — Installation, Hello World, the compiler toolchain.
  3. Basic Syntax — Comments, literals, operators, expressions, statements.
  4. Variables and Bindings — let, mut, type inference, shadowing.
  5. Types — Primitives, type annotations, type inference, custom types.
  6. Functions — Parameters, return types, implicit returns, closures, pipes.
  7. Control Flow — if, match, loops (for, while, loop), break/continue.
  8. Structs and Enums — Defining types, methods with impl, pattern matching.
  9. Pattern Matching — Match expressions, destructuring, guards.
  10. Error Handling — Option, Result, pattern matching on errors.
  11. Traits and Generics — Defining traits, implementing, generic functions.
  12. Modules and Organization — Import system, project structure.
  13. Building and Testing — Compilation, running tests, benchmarking.
  14. The Standard Library — Built-in functions, common patterns.

This book is for new and experienced programmers alike. Each chapter builds on the previous, so reading in order is recommended. Code examples are included throughout to illustrate concepts in action.

The full source code for Midori and this book is available on GitHub.

Midori is free and open source. Contributions welcome.
