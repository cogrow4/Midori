Chapter 4: Variables and Bindings

let Bindings

Variables in Midori are introduced with the let keyword. By default, every binding is immutable — once a value is assigned, it cannot change. This is a deliberate design choice: immutable data is easier to reason about, safer in concurrent contexts, and enables compiler optimizations.

let name = "Midori"
let year = 2026

// This would cause a compile error:
// name = "Something else"

Mutable Bindings with mut

When you need mutability, add the mut keyword after let. Mutable variables can be reassigned with =. Use mutability sparingly — prefer immutable bindings unless mutation is clearly the right tool.

let mut counter = 0
counter = counter + 1     // OK — counter is mutable
println(counter)           // 1

Type Inference

Midori has full type inference. You rarely need to write types explicitly — the compiler deduces them from usage. This gives you the safety of static types with the brevity of dynamic languages.

let x = 42            // inferred as Int
let y = 3.14          // inferred as Float
let z = "hello"       // inferred as Str

// You can annotate explicitly when needed
let n: Int = 42

Shadowing

Midori allows variable shadowing: you can declare a new variable with the same name as an existing one, and the new binding takes precedence. This is useful for transforming values without creating many distinct names.

let name = "midori"
let name = uppercase(name)  // shadows the previous binding
println(name)                // "MIDORI"

Shadowing is different from mutation. With shadowing, you create a new binding — the old value still exists but is no longer accessible. This lets you change the type of a variable, which mutation cannot do.

let value = "123"
let value: Int = parse_int(value)  // rebind to a different type

Constants

For values that are truly constant and known at compile time, use const. Constants must be annotated with their type and cannot be shadowed. They are inlined at every use site, making them suitable for configuration values.

const Int MAX_RETRIES = 3
const Str APP_NAME    = "Midori"
