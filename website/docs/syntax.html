Syntax

Comments

Midori supports line comments and block comments:

// This is a line comment

/* This is a block comment
   that spans multiple lines */

/* Comments can be /* nested */ inside each other */

String Interpolation

Strings support interpolation with curly braces:

let name = "Midori"
let version = 1
let msg = "Welcome to {name}, v{version}.{version + 1}!"

Variable Binding

Use let to bind values. Variables are immutable by default; use mut for mutable bindings:

let x = 42                // immutable
let mut y = 10             // mutable
y = y + 5

// Type annotations are optional
let z: Int = 7

Operators

Midori supports standard arithmetic and comparison operators:

// Arithmetic
let sum = 10 + 5      // addition
let diff = 10 - 5     // subtraction
let prod = 10 * 5     // multiplication
let quot = 10 / 5     // division
let rem = 10 % 3      // modulo

// Comparison
let eq = 5 == 5       // equal
let ne = 5 != 3       // not equal
let lt = 3 < 5        // less than
let gt = 5 > 3        // greater than
let le = 5 <= 5       // less than or equal
let ge = 5 >= 3       // greater than or equal

// Logical
let and = true && false  // logical and
let or = true || false   // logical or
let not = !true           // logical not

Pipe Operator

The pipe operator |> passes a value as the last argument to a function, enabling clean chaining:

let result = "hello"
    |> capitalize()
    |> greet()
    |> println()

// Equivalent to:
println(greet(capitalize("hello")))

Ranges

The .. operator creates an inclusive range:

let range = 0 .. 10  // 0, 1, 2, ..., 10

for i in 1 .. 5 {
    println(i)
}
// Prints 1, 2, 3, 4, 5

Semicolons and Newlines

Semicolons are optional. Newlines act as statement separators:

let a = 1
let b = 2

// You can use semicolons to put multiple statements on one line
let x = 1; let y = 2; let z = 3
