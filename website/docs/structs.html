Structs & Enums

Structs

Define a struct with the type keyword — a single variant with the same name as the type creates a struct:

type Point {
    x: Float
    y: Float
}

let p = Point { x: 1.0, y: 2.0 }
println(p.x)  // field access: 1.0

Enums

Multiple variants create an enum. Variants can hold data:

type Status {
    Active
    Inactive
    Banned { reason: Str }
}

// Enum field access via match
match status {
    Status::Active => println("active"),
    Status::Inactive => println("inactive"),
    Status::Banned { reason } => println("banned: {reason}"),
}

Methods with impl

Add methods to a type using impl. The first parameter is the receiver (conventionally named this):

type Vec2 {
    x: Float
    y: Float
}

impl Vec2 {
    fn len(this) -> Float {
        sqrt(this.x * this.x + this.y * this.y)
    }
}

let v = Vec2 { x: 3.0, y: 4.0 }
println(str_float(v.len()))  // 5.0

Traits

Traits define shared behavior across types:

trait Shape {
    fn area(this) -> Float
}

type Circle {
    radius: Float
}

impl Shape for Circle {
    fn area(this) -> Float {
        pi() * this.radius * this.radius
    }
}

Struct Field Access

Use dot notation to access fields, and methods via method dispatch:

let p = Point { x: 3.0, y: 4.0 }
println(str_float(p.x))  // field access
println(str_float(v.len()))   // method call
