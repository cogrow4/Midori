Control Flow

If / Else

Midori's if is an expression — it returns a value. No ternary operator needed:

let x = 10

if x > 0 {
    println("positive")
} else if x < 0 {
    println("negative")
} else {
    println("zero")
}

// if as an expression
let result = if x > 5 { "big" } else { "small" }
println(result)  // "big"

Match

Pattern matching with match is a powerful alternative to long if/else chains. Every arm is an expression:

match x {
    0 => println("zero"),
    1 => println("one"),
    2 .. 10 => println("between two and ten"),
    _ => println("something else"),
}

// Matching on enums
type Color {
    Red,
    Green,
    Blue,
    Custom(r: Int, g: Int, b: Int),
}

let color = Custom(255, 0, 0)
match color {
    Red => println("red"),
    Custom(r, g, b) => {
        println("rgb({r}, {g}, {b})")
    },
    _ => println("some color"),
}

// Matching on Option
let value: Option[Int] = Some(42)
let description = match value {
    Some(n) if n > 100 => "large",
    Some(n) => "number {n}",
    None => "nothing",
}

For Loops

Iterate over ranges, collections, and iterators:

// For loop with a range
for i in 0 .. 5 {
    println(i)
}
// Prints 0, 1, 2, 3, 4, 5

// Iterating over a list
let names = ["Alice", "Bob", "Charlie"]
for name in names {
    println(name)
}

// With index (destructuring)
for (i, name) in names.enumerate() {
    println("{i}: {name}")
}

While Loops

Loop while a condition holds:

let mut i = 0
while i < 5 {
    println(i)
    i = i + 1
}
// Prints 0, 1, 2, 3, 4

Loop (Infinite)

The loop keyword creates an infinite loop, useful with break for game loops or event-driven code:

let mut counter = 0
loop {
    println(counter)
    counter = counter + 1
    if counter >= 3 {
        break
    }
}
// Prints 0, 1, 2

Break and Continue

Control the flow inside loops:

// break — exit the loop early
for i in 0 .. 100 {
    if i * i > 50 {
        break
    }
    println(i)
}

// continue — skip to the next iteration
for i in 0 .. 10 {
    if i % 2 == 0 {
        continue
    }
    println(i)  // prints odd numbers only
}

// loop can return a value with break
let found = loop {
    let input = read_line()
    if input == "quit" {
        break "goodbye"
    }
}
