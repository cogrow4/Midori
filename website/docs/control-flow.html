Control Flow

If / Elif / Else

Conditional branching with if, elif, and else. Blocks are expressions — the last value is returned:

let x = 42
if x > 100 {
    println("big")
} elif x > 10 {
    println("medium")
} else {
    println("small")
}

// If as an expression:
let label = if x > 0 { "positive" } else { "non-positive" }

Match

Pattern matching with match — each arm uses => syntax:

match x {
    0 => println("zero"),
    1 => println("one"),
    _ => println("other"),
}

// Match as an expression:
let desc = match x {
    0 => "none",
    1 => "one",
    _ => "many",
}

// Matching on enums:
type Color {
    Red
    Green
    Blue
}

match color {
    Color::Red => println("red"),
    Color::Green => println("green"),
    _ => println("other"),
}

For Loops

Iterate over ranges and arrays with for...in:

// Range from 0 to 4
for i in range(5) {
    println(str(i))
}
// Prints 0, 1, 2, 3, 4

// Range with start and end (exclusive)
for i in range(2, 5) {
    println(str(i))
}
// Prints 2, 3, 4

// Iterate over array
let names = ["Alice", "Bob", "Charlie"]
for name in names {
    println(name)
}

While Loops

Loop while a condition holds:

var i = 0
while i < 5 {
    println(str(i))
    i = i + 1
}
// Prints 0, 1, 2, 3, 4

Loop / Break / Continue

loop creates an infinite loop. Use break to exit and continue to skip to the next iteration:

var i = 0
loop {
    i = i + 1
    if i % 2 == 0 {
        continue
    }
    if i > 5 {
        break
    }
    println(str(i))
}
// Prints 1, 3, 5
