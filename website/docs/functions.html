Functions

Functions are the fundamental building block of Midori programs. Use fn to define one.

Basic Syntax

fn greet(name: Str) -> Str {
    "Hello, {name}!"
}

fn main() {
    println(greet("Midori"))
}

Implicit Return

The last expression in a function body is automatically returned:

fn add(x: Int, y: Int) -> Int {
    x + y   // implicitly returned
}

fn factorial(n: Int) -> Int {
    if n <= 1 {
        1                   // block expression returns
    } else {
        n * factorial(n - 1)
    }
}

Return Type Annotation

Return types are specified with -> after the parameter list. Omitting -> defaults to Nil:

fn greet(name: Str) -> Str {
    "Hello, {name}!"
}

fn log(msg: Str) {
    println(msg)
    // implicitly returns Nil
}

Pipe Operator

The |> operator threads a value through a function call from left to right:

fn add_exclamation(s: Str) -> Str {
    s + "!"
}

// Without pipe:
println(add_exclamation("hello"))

// With pipe:
"hello" |> add_exclamation() |> println()

Recursion

Functions can call themselves recursively. Midori does not yet guarantee tail-call optimization:

fn factorial(n: Int) -> Int {
    if n <= 1 {
        1
    } else {
        n * factorial(n - 1)
    }
}

println(str(factorial(10)))  // 3628800
