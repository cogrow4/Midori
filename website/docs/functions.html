Functions

Defining Functions

Use the fn keyword to define a function. Parameters are annotated with types, and the return type follows ->:

fn greet(name: Str) -> Str {
    "Hello, {name}!"
}

fn main() {
    println(greet("Midori"))
}

Implicit Return

The last expression in a function body is automatically returned. No return keyword needed:

fn add(x: Int, y: Int) -> Int {
    x + y   // implicitly returned
}

fn factorial(n: Int) -> Int {
    if n <= 1 {
        1                   // early return via expression
    } else {
        n * factorial(n - 1)  // recursive call
    }
}

Parameters and Default Values

Functions can have default parameter values:

fn greet(name: Str, greeting: Str = "Hello") -> Str {
    "{greeting}, {name}!"
}

println(greet("Midori"))         // "Hello, Midori!"
println(greet("Midori", "Hi"))  // "Hi, Midori!"

Anonymous Functions (Closures)

Define inline functions with fn without a name. They can capture variables from their enclosing scope:

let double = fn(x: Int) -> Int { x * 2 }
println(double(5))  // 10

// Short form for simple closures
let triple = fn(x) { x * 3 }

// Capturing variables
let factor = 10
let multiply = fn(x: Int) { x * factor }

// Passing closures to higher-order functions
let numbers = [1, 2, 3, 4, 5]
let doubled = numbers.map(fn(x) { x * 2 })

Pipe Operator

The |> operator threads a value through a function call, making chains more readable:

fn clean(s: Str) -> Str { s.trim().lowercase() }
fn wrap(s: Str) -> Str { "[{s}]" }

// Without pipe — nested calls read inside-out
println(wrap(clean("  Hello  ")))

// With pipe — left-to-right, top-to-bottom
"  Hello  "
    |> clean()
    |> wrap()
    |> println()

// Output: [hello]

Method Calls

Types can have methods defined via impl blocks. Method calls use dot notation:

type Vec2 {
    x: Float,
    y: Float,
}

impl Vec2 {
    fn length(self) -> Float {
        (self.x * self.x + self.y * self.y).sqrt()
    }

    fn add(self, other: Vec2) -> Vec2 {
        Vec2 { x: self.x + other.x, y: self.y + other.y }
    }
}

let v = Vec2 { x: 3.0, y: 4.0 }
println(v.length())  // 5.0

Functions as Values

Functions are first-class values and can be passed around:

fn apply_twice(f: fn(Int) -> Int, x: Int) -> Int {
    f(f(x))
}

fn increment(n: Int) -> Int { n + 1 }

println(apply_twice(increment, 5))  // 7
