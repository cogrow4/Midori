Types

Midori is a statically typed language with type inference. Most of the time you don't need to write types — the compiler infers them.

Primitive Types

let count: Int = 42              // 64-bit signed integer
let pi: Float = 3.14              // 64-bit floating point
let is_active: Bool = true     // boolean (true/false)
let msg: Str = "hello"           // string
let ch: Char = 'A'                // ASCII character

Type Inference

Type annotations are optional — the compiler infers types from values:

let x = 42               // x is Int
let y = "hello"           // y is Str
let z = 3.14              // z is Float
let b = true               // b is Bool

Function Types

Functions have types too. A function that takes an Int and returns a Str has type Fn(Int) -> Str.

fn greet(n: Int) -> Str {
    "Number: {n}"
}
// greet has type: Fn(Int) -> Str

Custom Types

Define structs and enums with the type keyword:

type Point {
    x: Float
    y: Float
}

type Color {
    Red
    Green
    Blue
}

type Shape {
    Circle { radius: Float }
    Rect { width: Float, height: Float }
}

See the Structs & Enums page for details on custom types and methods.

Type Annotations

Use : after the variable name to annotate a type:

let name: Str = "Midori"
let age: Int = 1
let pi: Float = 3.14159
