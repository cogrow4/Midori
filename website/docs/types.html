Types

Midori is a strongly typed language with full type inference. You rarely need to write types explicitly — the compiler can infer them from context.

Primitive Types

Int

A 64-bit signed integer:

let a: Int = 42
let b = -17           // inferred as Int
let hex = 0xFF        // hexadecimal literal
let bin = 0b1010      // binary literal

Float

A 64-bit floating point number:

let pi: Float = 3.14159
let e = 2.71828        // inferred as Float
let sci = 1.5e10      // scientific notation

Bool

A boolean value, either true or false:

let is_ready: Bool = true
let done = false        // inferred as Bool

Str

A UTF-8 string with interpolation support:

let name = "Midori"
let greeting = "Hello, {name}!"  // "Hello, Midori!"

// Multi-line strings
let poem = "Roses are red,
Violets are blue,
Midori is fresh,
And so are you."

Char

A single Unicode character:

let letter: Char = 'A'
let emoji = '🍃'

Generic Types

Option[T]

Represents a value that may or may not be present. Eliminates null pointer exceptions:

let some_value: Option[Int] = Some(42)
let no_value: Option[Int] = None

// Pattern matching on Option
match some_value {
    Some(val) => println("Got {val}"),
    None => println("No value"),
}

Result[T, E]

Represents either a success value or an error. Enforces error handling at the type level:

fn divide(a: Int, b: Int) -> Result[Int, Str] {
    if b == 0 {
        Err("division by zero")
    } else {
        Ok(a / b)
    }
}

let result = divide(10, 2)
match result {
    Ok(val) => println("Result: {val}"),
    Err(msg) => println("Error: {msg}"),
}

Type Annotations

While Midori infers types, you can annotate them explicitly for clarity or when the compiler needs help:

fn add(x: Int, y: Int) -> Int {
    x + y
}

let items: List[Str] = List.new()
let pair: (Str, Int) = ("age", 30)

Type Inference

The compiler deduces types from how values are used:

let x = 42               // x is Int
let y = 3.14              // y is Float
let z = x + to_int(y)      // z is Int (y converted)

// Generic functions infer their type parameters
fn identity[T](val: T) -> T { val }
let n = identity(42)      // n is Int
let s = identity("hi")    // s is Str
