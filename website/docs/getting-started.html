Getting Started

Installation

Download the latest Midori compiler from GitHub. Pre-built binaries are available for macOS, Linux, and Windows.

Once downloaded, add the midori binary to your $PATH and verify the installation:

$ midori --version
midori 0.1.0

Your First Program

Create a file called hello.mi:

// hello.mi
fn main() {
    println("Hello, world!")
}

Running the Program

Use the midori run command to execute it directly:

$ midori run hello.mi
Hello, world!

Or use midori build to compile an executable:

$ midori build hello.mi
$ ./hello
Hello, world!

Variables and Basic Types

Midori is immutable by default. Use let to bind a value, and mut to make it mutable:

let greeting = "Hello"          // immutable
let mut count = 0            // mutable
count = count + 1

// Type annotation (optional — Midori infers types)
let name: Str = "Midori"
let age: Int = 1

Basic Types

Midori provides a familiar set of primitive types:

let count: Int = 42              // 64-bit signed integer
let pi: Float = 3.14              // 64-bit floating point
let is_active: Bool = true     // boolean
let msg: Str = "Hello, {name}"  // string with interpolation
let ch: Char = 'A'                // character

Printing Output

Use the built-in println function to print to stdout:

fn main() {
    let name = "Midori"
    let year = 2026
    println("Welcome to {name}, version {year}!")
}

// Output: Welcome to Midori, version 2026!
