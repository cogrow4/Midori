Midori Language Specification

Version: 0.1.0 (pre-release)

1. Lexical Structure

1.1 Keywords

fn  func  let  var  mut  if  else  elif  match  for  while  loop
break  continue  return  type  trait  impl  pub  import  as
extern  true  false  nil  in  and  or  not  self  where

1.2 Literals

  • Integer: 42 — 64-bit signed
  • Float: 3.14 — 64-bit IEEE 754
  • String: "hello" — with interpolation: "Hello, {name}!"
  • Char: 'A' — ASCII
  • Bool: true / false
  • Nil: nil

2. Programs

A Midori source file (.mi) consists of a sequence of declarations and statements. Top-level code that is not a declaration (func, type, trait, impl, import, extern) is implicitly wrapped in a func main() { } function.

// This is a complete program — no main() needed
print("Hello, World!")

3. Variables

let creates an immutable binding. var creates a mutable binding.

let x = 42          // immutable
var y = 99          // mutable
y = y + 1

// Explicit type annotation
let name: Str = "Midori"

4. Functions

Functions are defined with func (or fn for backwards compatibility):

func greet(name: Str) {
    println("Hello, {name}!")
}

func add(x: Int, y: Int) -> Int {
    x + y  // implicit return
}

Parameters can have type annotations. Return type is specified with -> if not inferred.

5. Control Flow

5.1 If / Elif / Else

if x > 0 {
    println("positive")
} elif x < 0 {
    println("negative")
} else {
    println("zero")
}

5.2 Match

Pattern matching uses : between the pattern and its body:

match x {
    1: println("one")
    2: println("two")
    _: println("other")
}

5.3 For

for i in range(5) {
    println(str(i))
}

for name in ["A", "B"] {
    println(name)
}

5.4 While

var i = 0
while i < 5 {
    i = i + 1
}

5.5 Loop

loop {
    if done { break }
    continue
}

6. Types

6.1 Primitive Types

  • Int — 64-bit signed integer
  • Float — 64-bit floating point
  • Bool — boolean (true/false)
  • Str — string with length prefix
  • Char — ASCII character
  • Nil — unit type (void)

6.2 Custom Types

type Point {
    x: Float
    y: Float
}

type Color {
    Red
    Green
    Blue
}

7. Methods

type Vec2 {
    x: Float
    y: Float
}

impl Vec2 {
    func len(this) -> Float {
        sqrt(this.x * this.x + this.y * this.y)
    }
}

let v = Vec2 { x: 3.0, y: 4.0 }
println(str_float(v.len()))
