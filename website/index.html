Why Midori?
🌿

Beautiful Syntax

Clean, expressive syntax that feels natural. Type inference, pattern matching, and string interpolation make code a pleasure to write. Familiar C-style braces with Pythonic sensibilities.

⚡

Compiled & Fast

Compiles through C to native machine code via Clang/GCC. Zero-cost abstractions mean you never pay for what you don't use. No runtime overhead, no garbage collector pauses.

🔒

Type Safe

Strong static typing with full type inference. No null pointer exceptions. Option and Result types make error handling explicit and compiler-enforced.

🔄

Immutable by Default

Variables are immutable by default. The var keyword makes mutability explicit, and the type system tracks it. Makes code easier to reason about and parallelize.

🎯

Developer UX First

From the error messages to the CLI toolchain, every surface is designed to keep you in flow state. One binary, zero dependencies, instant startup.

🔄

Self-Hosting Path

The Midori compiler is bootstrapping-ready. Written in Rust first, the plan is to rewrite the compiler in Midori itself — proving the language is production-ready.

Quick Start
# Install
curl -fsSL https://midori-lang.dev/install.sh | sh

# Create and run
midori new my-project
cd my-project
midori run main.mi

# Or just run a single file
midori hello.mi # Hello, Midori!
