Compiled, Pythonic, from scratch

Midori is a compiled programming language with Python-inspired syntax and C-style braces. Type inference, string interpolation, match expressions, method dispatch — built from the ground up in Rust. Compiles through C to native code.

// hello.mi
fn greet(name: Str) -> Str {
  "Hello, {name}!"
}

fn main() {
  println(greet("Midori"))
}
